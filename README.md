A simple investment calculator that calculates investments with and without monthly deposits.

When calculating without monthly deposits, compounding and contributions occur each year.
When calculating with monthly deposits, compounding and contributions occur at the start of each month.

How to run:
1. Compile the code. If you use g++, that would look like this:
    g++ src/main.cpp src/investment_calculator.cpp -o InvestmentCalculator
2. Run the compiled file.
